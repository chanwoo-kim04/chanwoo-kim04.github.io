**Solution:** Let $X = \mathbb{N}$. 
For each positive integer $n$, define the partition $\mathcal{P}_n$ of $X$ as
$$
  \mathcal{P}_n = \qty{\qty{1}, \qty{2}, \dots, \qty{n}, \qty{n+1, n+2, \dots}}
$$
Let $\mathcal{A}_n$ be the $\sigma$-algebra generated by $\mathcal{P}_n$. 
Since $\mathcal{P}_n$ is a finite partition, $\mathcal{A}_n$ consists of all unions of sets in $\mathcal{P}_n$.  
By construction, $\mathcal{A}_1 \subset \mathcal{A}_2 \subset \cdots$ is an increasing sequence of $\sigma$-algebras. 

Let $\mathcal{A} = \bigcup_{n=1}^{\infty} \mathcal{A}_n$.
We will show that $\mathcal{A}$ is not a $\sigma$-algebra by finding a countable collection of sets in $\mathcal{A}$ whose union is not in $\mathcal{A}$. 

Consider the sets $S_n = \qty{2n}$ for $n \in \mathbb{N}$. 
For any given $n$, the set $S_n$ is an element of the partition $\mathcal{P}\_{2n}$, and thus $S_n \in \mathcal{A}\_{2n}$. 
Since $\mathcal{A}\_{2n} \sub \mathcal{A}$, it follows that $S_n \in \mathcal{A}$ for all $n$. 

Now, consider the countable union of these sets:
$$
  S = \bigcup_{n=1}^{\infty} S_n = \qty{2, 4, 6, 8, \dots}
$$
This is the set of all even natural numbers. We will now show that $S \notin \mathcal{A}$. 

Assume for contradiction that $S \in \mathcal{A}$. 
By the definition of $\mathcal{A}$, there must exist some finite integer $k$ such that $S \in \mathcal{A}_k$. 
Any set in $\mathcal{A}_k$ is a union of elements from the partition $\mathcal{P}_k = \qty{\qty{1}, \dots, \qty{k}, \qty{k+1, k+2, \dots}}$. 
Let $U_k = \qty{k+1, k+2, \dots}$. 
A set in $\mathcal{A}_k$ is either a subset of $\qty{1, \dots, k}$ or it contains $U_k$ as a subset. 

The set $S$ is infinite, so if $S$ were in $\mathcal{A}_k$, it would have to contain $U_k$. 
This is a contradiction, as $S$ contains no odd numbers, while the set $U_k$ contains infinitely many. 

Thus, $S \notin \mathcal{A}_k$ for any $k$, which means $S \notin \mathcal{A}$. 
We have found a countable collection of sets $\qty{S_n}$ in $\mathcal{A}$ whose union $S$ is not in $\mathcal{A}$. 
This shows that $\mathcal{A}$ is not closed under countable unions and is therefore not a $\sigma$-algebra.
